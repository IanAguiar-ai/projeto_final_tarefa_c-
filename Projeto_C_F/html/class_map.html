Detailed Description

A função "proxima_fase" basicamente cuida do processo de geração de novos mapas, esses mapas tem sempre tamanhos difetentes e neles são adicionados os obstaculos e as joias, quanto maior o nível da fase, mais objetos ele adiciona.

OBJ1O "OBJ1" é uma instancia da classe Robots, em outras palavras é o jogados, ele entra nessa função para que a geração de objetos não sobreponha ele no mapa.
